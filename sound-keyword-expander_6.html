// 支援的平台
C
Anthropic Claude（本工具使用）
登入 → console.anthropic.com/settings/keys
點「Create Key」→ 複製以 sk-ant- 開頭的字串
G
Google Gemini
登入 → aistudio.google.com/app/apikey
點「Create API key」→ 複製以 AIza 開頭的字串
O
OpenAI (ChatGPT)
登入 → platform.openai.com/api-keys
點「Create new secret key」→ 複製以 sk- 開頭的字串
M
Mistral AI
登入 → console.mistral.ai/api-keys
點「Create new key」→ 複製產生的字串
⚠ 注意：API Key 就像密碼，不要分享給別人。
本工具的 Key 只存在你的瀏覽器本地（localStorage），不會傳送到任何伺服器。
使用 API 會依照各平台費率計費，Anthropic 新帳號通常有免費額度。
